Gift Card

Métodos para cartão presente.

Create

POST: https://zeloapi.herokuapp.com/giftcard/create

                                    {
                                        "card_id": "GC-00001",
                                        "message": "Mensagem cartão presente",
                                        "balance": "100.50",
                                        "email": "email@ezcommerce.com.br"
                                    }

Balance

POST: https://zeloapi.herokuapp.com/giftcard/balance

                                    {
                                        "card_id": "GC-00001",
                                        "email": "some value"
                                    }

Capture

POST: https://zeloapi.herokuapp.com/giftcard/capture
Para capturar um cartão, pode ser usado o id do cartão ou o email como filtro.
A captura pode ser feita com mais de um cartão. 

                                    {
                                        "order_number": "string",
                                        "cards": [
                                            {
                                                "card_id": "GC-00001",
                                                "email": "email@ezcommerce.com.br"
                                            }
                                        ]
                                    }

List

GET: https://zeloapi.herokuapp.com/giftcard/
Listar todos os cartões. 

                                    //retorno
                                    [
                                        {
                                            "card_id": "GC-00001",
                                            "message": "Mensagem cartão presente",
                                            "email": "email@ezcommerce.com.br",
                                            "is_valid": true,
                                            "balance": "100.50"
                                        }
                                    ]

Loyalty Card

Métodos para cartão fidelidade.

Create

POST: https://zeloapi.herokuapp.com/loyaltycard/create

                                    {
                                        "card_id": "LO-00001",
                                        "message": "Mensagem cartão fidelidade",
                                        "is_valid": true,
                                        "balance": "200",
                                        "email": "email@ezcommerce.com.br",
                                        "document_number": "74151037268"
                                    }

Balance

POST: https://zeloapi.herokuapp.com/loyaltycard/balance
Para pesquisar o valor disponivel no cartão.
Pode ser usado como filtro o card_id, document_number ou o email usado no create. 

                                    {
                                        "card_id": "some value",
                                        "document_number": "74151037268",
                                        "email": "some value"
                                    }

Capture

POST: https://zeloapi.herokuapp.com/loyaltycard/capture
Para capturar um cartão, pode ser usado o card_id, document_number ou o email usado no create.
Value é o valor que será capturado do cartão. 

                                    {
                                        "order_number": "123144123",
                                        "card": {
                                            "card_id": "some value",
                                            "document_number": "74151037268",
                                            "email": "some value"
                                        },
                                        "value": 160
                                    }

StatementHtml

POST: https://zeloapi.herokuapp.com/loyaltycard/statementHtml
Retorna o html do cartão.

                                    {
                                        "card_id": "some value",
                                        "document_number": "74151037268",
                                        "email": "some value"
                                    }

List

GET: https://zeloapi.herokuapp.com/loyaltycard/
Listar todos os cartões. 

                                    //retorno
                                    [
                                        {
                                            "card_id": "LO-00001",
                                            "message": "Mensagem cartão fidelidade",
                                            "email": "email@ezcommerce.com.br",
                                            "document_number": "74151037268",
                                            "is_valid": true,
                                            "balance": "200"
                                        }
                                    ]
